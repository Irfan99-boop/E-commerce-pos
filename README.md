# E-Commerce & Real-time POS App

Halo! Ini adalah proyek yang sedang saya kerjakan untuk mendalami ekosistem Full-stack JavaScript, khususnya menggunakan Next.js dan TypeScript. Aplikasi ini fokus pada sistem kasir (Point of Sale) yang bisa mengelola transaksi secara real-time.
Proyek ini dibangun sebagai bagian dari pengembangan keahlian Full-stack Developer.

### Kenapa saya buat proyek ini?

Saya ingin mencoba tantangan mengelola data yang kompleks, seperti sinkronisasi stok barang saat ada transaksi dan bagaimana menangani pembayaran otomatis menggunakan Midtrans.

## 🚀 Tech Stack

- **Frontend:** Next.js (App Router), TypeScript, Tailwind CSS, Shadcn UI
- **State Management:** Zustand
- **Data Fetching:** Tanstack React Query
- **Backend/Database:** Supabase
- **Validation:** React Hook Form & Zod
- **Payment Gateway:** Midtrans (Integration Stage)
- **Deployment:** Vercel

## ✨ Fitur Utama

- **Real-time Inventory:** Pembaruan stok barang secara otomatis saat transaksi terjadi.
- **Modern UI:** Antarmuka responsif yang nyaman digunakan di perangkat desktop maupun tablet.
- **Secure Transaction:** Validasi data yang ketat di sisi klien dan server.
- **Integrated Payment:** Persiapan sistem pembayaran otomatis menggunakan Midtrans.
